Perform Raspberry Pi Interfacing with EM18 RFID Reader

1. Open your “Raspberry pi Configuration” to enable serial Port & Disable Serial Console.

2. Select “Interfaces”:

Now Enable Serial Port and Disable Serial Console and click on OK

Click on Yes and Reboot the Raspberry pi.

Step 3] Verify if the Changes have been made. Open Terminal and Enter Command:
“ls -l /dev”

Here the “ttys0” is our port
Let’s send a message on this port 

Step 4] Open a new terminal and type command 
echo “Hello There”  > /dev/ttys0

Step 5] Disable the Bluetooth

Step 6] then disable it from Invoking from Restart for that in the terminal enter command:
sudo nano /boot/config.txt

Enter : “dtoverlay=disable-bt” at the bottom of the txt file and then save it 

Step 7] Then disable Bluetooth related services using command:
Sudo systemctl disable hciuart

After this Reboot raspberry pi again.
The bluetooth logo will not be visible 

Step 8] Now send the message using the other port

Serial_Read:

import serial
from time import sleep

ser = serial.Serial ("/dev/ttyAMA0", 9600)    #Open port with baud rate
while True:
    received_data = ser.read()              #read serial port
    sleep(0.03)
    data_left = ser.inWaiting()             #check for remaining byte
    received_data += ser.read(data_left)
    print (received_data)                   #print received data
    ser.write(received_data)                #transmit data serially
